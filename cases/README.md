# cases

这里存放具体的上下文学习的案例

每个案例尽量的保留：

- 原始待训练上下文
- 训练目标
- 训练数据集
- 标注记录
- 输出对比
- checkpoint
- 最终效果


# 基本结构

```text
案例名/
  README.md
  training-plan.md
  training-targets.md
  labels.md
  materials/
  outputs/
  checkpoint/
```

说明：

- `training-plan.md`：训练计划与训练流程。
- `training-targets.md`：训练目标。
- `labels.md`：人工标注与偏好记录。
- `materials/`：上下文训练过程中使用的材料。
- `outputs/`：模型输出记录。
- `checkpoint/`：案例内部的阶段性版本记录。
