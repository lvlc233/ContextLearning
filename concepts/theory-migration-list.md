# 理论迁移清单

这份清单只记录**可能进入上下文学习方法论**的内容。

判断标准：

- 是否直接讨论上下文内容如何被修改、组织、评估或迭代；
- 是否能解释多个上下文训练案例，而不是只解释某个 skill 的交互习惯；
- 是否能沉淀为概念、流程或测试方法，而不是单次经验；
- 是否有足够证据支撑进入正文。

不满足这些条件的内容，应归入 `cases/`、`$operator-knowledge` 或直接删除，不在本清单中继续推进。
