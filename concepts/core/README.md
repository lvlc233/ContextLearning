# 核心概念

这里存放上下文学习方法论中的基础概念。

## 导读

| 概念 | 简单含义 | 建议阅读 |
|---|---|---|
| `上下文学习` | 通过调整上下文信息,使模型在加载上下文后更稳定地逼近期望行为。 | [001-context-learning.md](001-context-learning.md) |
| `行为` | 上下文希望影响的可观察输出、判断、选择或执行方式。 | [002-行为.md](002-行为.md) |
| `上下文环境` | 上下文被加载、运行和产生效果时所处的模型、工具、任务与约束条件。 | [03上下文环境.md](03上下文环境.md) |
| `数据标注` | 用自然语言记录人类判断、偏好、修正意见或版本比较。 | [003-数据标注.md](003-数据标注.md) |
| `训练信号` | 运行、标注、测试或版本对比中产生的判断依据。 | [004-训练信号.md](004-训练信号.md) |
| `训练目标` | 希望上下文最终引发的可观察行为。 | [005-training-target.md](005-training-target.md) |
| `训练方向` | 判断一次输出或修改更接近目标还是更偏离目标的标准。 | [006-training-direction.md](006-training-direction.md) |
