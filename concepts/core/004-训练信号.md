# 训练信号

`训练信号` ，又称`训练反馈信号`、`反馈信号`，它指的是：在上下文训练过程中所产生的用于判断上下文是否正在逼近训练目标的信息。

它可以来自人类反馈、样本表现、失败案例、版本对比或测试结果。

训练信号不是训练目标本身，也不应该被原样塞进上下文。它的作用是帮助判断：当前输出或上下文修改，是更接近目标，还是更偏离目标，或者是其他什么效果。

## 与训练目标和训练方向的关系

如果说 `训练目标` 回答的是：

```text
我希望上下文最终引发什么行为？
```

`训练方向` 回答的是：

```text
什么更接近目标？
什么更偏离目标？
```

那么 `训练信号` 回答的是：

```text
当前这次输出、修改或测试结果，实际提供了什么判断依据？
```

三者之间的关系可以理解为：

```text
训练目标：期望逼近的行为。
训练方向：判断接近或偏离的标准。
训练信号：运行过程中观察到的判断依据。
```

训练信号通常需要被进一步转化为训练方向、反馈信息、规则约束或测试标准，而不是直接作为上下文本体的一部分。

## 偏离信号

`偏离信号` 是训练信号的一种，指模型输出、上下文修改或测试结果相对于训练目标、训练方向或人类标注出现偏差时，被观察到的证据。

偏离信号的重点不是“输出不好”，而是要指出：

- 偏离了哪个训练目标；
- 偏离发生在什么输出位置或行为上；
- 偏离更接近哪一种负向信号；
- 偏离是否稳定出现；
- 偏离是否足以触发上下文修改。

例如：

```text
训练目标：
压缩上下文时，保留任务边界、失败条件和测试标准。

模型输出：
新版本上下文更短，但删除了失败条件。

偏离信号：
压缩行为达成，但关键约束丢失。
```

这里的偏离信号不是“压缩得不好”这样一个笼统评价，而是指出当前输出相对于训练目标的具体差距。

## 正向信号与负向信号

训练信号可以表现为正向信号，也可以表现为负向信号。

`正向信号` 指当前输出或修改更接近训练目标。例如：

```text
新上下文删除了重复表达，同时保留了任务边界。
```

`负向信号` 指当前输出或修改更偏离训练目标。例如：

```text
新上下文更短，但删除了失败条件。
```

偏离信号通常属于负向信号，但并不完全等同于负向信号。负向信号可以是一个判断结果，偏离信号更强调这个判断背后的可观察证据。

## 反馈信息

`反馈信息` 是对训练信号的整理和表达。

偏离信号本身只是观察结果，还不能直接等同于上下文修改。训练时通常需要先把偏离信号整理成反馈信息，再根据反馈信息调整上下文。

例如：

```text
偏离信号：
新上下文更短，但删除了失败条件。

反馈信息：
当前压缩规则只强调减少重复表达，没有约束必要边界的保留。
```

反馈信息比偏离信号更接近上下文修改，但它仍然不是最终规则。最终是否修改上下文、如何修改上下文，还需要结合训练目标、训练方向和其他样本表现判断。

## 边界

训练信号不是案例堆积。

训练信号也不是把人类反馈逐字写入上下文。

一个训练信号是否有效，关键不在于它是否来自真实案例，而在于它是否能帮助判断上下文正在接近目标还是偏离目标。
