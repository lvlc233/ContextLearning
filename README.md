## 当前整理原则

- 主文档保留方法论主线。
- 案例独立出来，避免主方法论被具体案例拉偏。
- 案例材料放在案例内部，避免材料和方法论混在一起。
- 尚未稳定的方法流程先放入 notes，等待后续沉淀。
- checkpoint 单独放置，便于记录阶段性版本。
